Paragraphs

A paragraph (from the Greek paragraphos, “to write beside” or “written beside”) is a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.

Answers to Questions

  1. What is selector’s specificity and how it’s calculated?

    Selector specificity is a value that determines which CSS rule will be applied to an element when multiple rules target the same element. Specificity is calculated based on the types of selectors used in a CSS rule. Specifically:

    • Inline styles have the highest specificity.
    • ID selectors have higher specificity than class selectors, which have higher specificity than element selectors.
    • Universal selectors (*) have zero specificity.
    • Specificity is calculated by assigning a numerical value to each type of selector in a rule, and the rule with the highest specificity wins.

  2. What are complex selectors?

    Complex selectors are CSS selectors that combine multiple simple selectors to target elements with greater specificity. Examples of complex selectors include:

    • Descendant selectors (e.g., div p) target elements that are descendants of another element.
    • Child selectors (e.g., div > p) target elements that are direct children of another element.
    • Adjacent sibling selectors (e.g., h2 + p) target elements that are immediately preceded by a sibling element.
    • General sibling selectors (e.g., h2 ~ p) target elements that are siblings of another element.

  3. What is the difference between pseudo-classes and pseudo-elements?

    Pseudo-classes and pseudo-elements are both special keywords that can be added to selectors to style elements in specific states or parts of an element's content. The main difference between them is:

    • Pseudo-classes select elements based on their state or position in the document (e.g., :hover, :nth-child()).
    • Pseudo-elements select parts of an element's content (e.g., ::before, ::after) or generate content that is not present in the HTML markup.

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.

It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.
