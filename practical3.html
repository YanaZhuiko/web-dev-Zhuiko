1. How to open the developer console in Mozilla Firefox and Safari?

Mozilla Firefox: In Firefox, you can open the developer console by pressing Ctrl+Shift+K (Windows/Linux) or Cmd+Option+K (Mac).

Safari: To open the developer console in Safari, you can use the following methods:

  • Press Option+Command+C.
  • Alternatively, you can enable the Develop menu in Safari's preferences under the Advanced tab. Once enabled, you can access the developer console by navigating to Develop > Show Error Console or Develop > Show JavaScript Console.

2. What is the preferable way of including internal links into an HTML page: absolute or relative links? Describe use cases for choosing absolute links and choosing relative links.

Absolute Links: These links contain the full URL of the linked page, including the protocol (e.g., http://) and domain (e.g., example.com). They are useful when linking to external websites or specific resources on other domains.

Relative Links: Relative links specify the path to the linked page relative to the current page's location. They are preferred for internal links within the same website because:

  • They are more flexible and maintainable, especially when restructuring a website's directory structure or moving content to a different location.
  • They are shorter and cleaner compared to absolute URLs, making the HTML markup more readable.

Use cases:

  • Absolute Links: Useful for linking to external resources or pages on other domains where the full URL is necessary.
  • Relative Links: Preferred for internal navigation within the same website. For example, linking between pages within the same website's structure or linking to images, stylesheets, or scripts stored in the same directory or subdirectory.

3. Research and write about image formats used in web development. What is the difference between JPG, GIF, and PNG formats? What is WEBP image type? What is SVG?

JPG (JPEG): JPEG is a lossy compression format commonly used for photographs and images with complex colors and gradients.

GIF: GIF is a lossless compression format that supports animations and transparency.

PNG: PNG is a lossless compression format suitable for images with sharp edges, text, or transparency.

WEBP: WEBP is a modern image format developed by Google, offering both lossy and lossless compression.

SVG: SVG is a vector graphics format based on XML, ideal for scalable graphics and interactive elements.
